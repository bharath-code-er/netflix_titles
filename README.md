# 📊 Netflix Movies & TV Shows – Exploratory Data Analysis

## 📌 Overview

This project performs **Exploratory Data Analysis (EDA)** on the Netflix Movies & TV Shows dataset.
The goal is to analyze, clean, and visualize the dataset to discover useful insights about Netflix content.

---

## 🛠️ Tools & Technologies

* Python 🐍
* Pandas
* NumPy
* Matplotlib
* Seaborn

---

## 🔍 Project Workflow

### 1. Data Cleaning

* Handled missing values (director, cast, country, rating)
* Removed duplicate records
* Dropped rows with missing `date_added`

### 2. Data Preprocessing

* Converted `date_added` to datetime format
* Extracted `year_added`
* Converted `duration` into numeric format

### 3. Data Visualization

* Movies vs TV Shows distribution
* Content added over the years
* Top 10 content-producing countries
* Ratings distribution
* Duration distribution

---

## 📈 Key Insights

* Netflix has more **Movies** than TV Shows
* Content increased rapidly after **2015**
* **USA and India** are the top content producers
* Most content is targeted toward **mature audiences (TV-MA, TV-14)**
* Movie durations are mostly between **80–120 minutes**
